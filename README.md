# Where's My Fluffy
a full-stack web platform for pet location tracking using Java Spring Boot, Vue.js, and Tailwind CSS.

# run backend

brew install maven
```
cd backend
./mvnw spring-boot:run
```
back-end visit http://localhost:8080

# run frontend

```
cd frontend
npm install
npm run dev
```
front-end visit http://localhost:5173
